Krita 4.2.1
June 5, 2019
Krita 4.2.1 Apenas unos días después de lanzar la versión 4.2.0 ponemos a disposición la 4.2.1. En ésta se ha resuelto el problema donde después de usar Krita por un tiempo, el dibujar se hace bastante lento, esto por que las entradas de “deshacer” se hacían infinitas.
Se da de baja el sitio ask.krita.org
June 5, 2019
Hace un año, mas o menos, creamos el sitio web ask.krita.org. La idea era el tener un espacio similar a stack-exchange, donde los usuarios pudieran buscar si alguien mas tenia el mismo problema que ellos, ver si había soluciones a los mismos y si no, podían reportarlos.
