Krita 3.1.3 versión alfa

31 de marzo de 2017 | Reading time: 2 min

Tras un arduo trabajo en el desarrollo de Krita, finalmente la versión alfa de prueba 3.1.3 ha sido lanzada. Ésta versión sera principalmente para la corrección de errores de código, nuevas funciones se esperan solamente en la versión 4.0. Krita 3.1.3 estable, está programada para lanzarse al final de Abril.

Aún hay bastante que corregir en Krita alfa, por lo que esperamos que nuestros usuarios descarguen y la pongan a prueba reportándonos cualquier problema que encuentren, para verificar si ya se han reportado puedes buscar en la lista de errores de código.

Nota para los usuarios de Windows

Todavía existen problemas con los controladores de video de Intel, las ultimas actualizaciones de Windows parecen haber descompuesto el OpenGL ne Krita, pero dado a que los desarrolladores no poseen ninguna computadora con el sistema problemático específicamente, no ha sido posible indagar en cual es la verdadera causa de estos errores.

Windows

En Windows, si se encuntran con cierres del programa inesperados, favor de seguir las siguientes instrucciones (en ingles) para encontrar los errores y así poder entender por que suceden.

  • 32 bits Windows: krita-3.1.3-alpha.2-x86-setup.exe

  • Portable 32 bits Windows: krita-3.1.3-alpha.2-x86.zip

  • Debug symbols. (Se habre en el archivo donde se instala Krita)

  • 64 bits Windows: krita-3.1.3-alpha.2-x64-setup.exe

  • Portable 64 bits Windows: krita-3.1.3-alpha.2-x64.zip

  • Debug symbols. (Se habre en el archivo donde se instala Krita)

Linux

  • 64 bits Linux: krita-3.1.3-alpha.2-x86_64.appimage

En Ubuntu y sus derivados puedes usar Krita Lime PPA para instalar Krita 3.1.3-alpha.2, se espera un paquete Snap para ésta versión muy pronto.
